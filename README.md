# 📊 E-commerce Customer Churn Analysis

## 🔍 Project Overview

This project analyzes customer churn behavior using a dataset of 50,000 customers. The goal is to identify patterns and factors affecting customer retention.

---

## 🛠 Tools Used

* Excel (Data Cleaning)
* Python (Data Analysis)
* Power BI (Dashboard Visualization)

---

## 📈 Key Insights

* Customers with low login frequency have the highest churn rate
* Inactive users are more likely to leave the platform
* Higher engagement leads to better customer retention

---

## 📊 Dashboard Features

* Churn Rate KPI
* Login vs Churn Analysis
* Recency vs Churn Analysis
* Country-wise Churn Insights

---

## 🎯 Conclusion

Customer engagement plays a crucial role in reducing churn. Businesses should focus on increasing user activity and targeting inactive customers.
